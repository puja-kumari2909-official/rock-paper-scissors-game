# Rock–Paper–Scissors Game

A clean and simple Rock–Paper–Scissors game built using **HTML, CSS, and JavaScript**.  
The user selects an option, the computer generates a random choice, and the game displays the result instantly.

## Features
- Simple and responsive UI  
- Random computer moves  
- Instant win/lose/draw result  
- Beginner-friendly code structure

## Tech Used
- HTML  
- CSS  
- JavaScript  

## How to Run
Open the **index.html** file in your browser to play the game.

## Project Files
- `index.html`  
- `style.css`  
- `app.js`
